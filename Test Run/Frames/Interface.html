Interface

Milliseconds Per FrameDisplays the time taken to calculate and render each frame of the simulation
Altitude MeterDisplays the Altitude, Rate of change and Vertical Acceleration
Angle of Attack & Lift IndicatorDisplays the Angle of Attack and Lift
Artificial Horizon Displays the Attitude- the elevation from the horizontal, the compass direction and the roll
Selected Target Highlights the current target on screen, or points towards it if it is off screen
Targeting Computer Shows the point of aim to hit the current target, assuming constant acceleration
Targeting Reticle Shows the direction in which any shots fired will fly
Target InformationDisplays the current target's Attitude, Distance, Hitpoints and Objective
Throttle Level The current Throttle level
Afterburner Level The current Afterburner level; turns yellow if Afterburners are active
Air Speed Indicator & Accelerometer Displays the Air Speed and Acceleration
Weapons List Lists the available weapons, their current state of reloading. The currently selected weapon is highlighted. Weapons which cannot reach the current target are shown yellow
