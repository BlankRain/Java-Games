Controls

Aircraft Controls

Control of the aircraft is mostly done with the mouse, which substitutes for a joystick. Left and right mouse controls roll, while up and down mouse controls the pitch of the aircraft.
Mouse X Roll Speed (Ailerons)
Mouse Y Pitch (Elevators)
Left Click Switch Weapon
Right Click Center Mouse
Scroll Wheel Throttle
Shift Afterburners
Spacebar Fire Weapon
Shift Afterburners

Targeting

These controls allow you to select which unit you want to target. This allows you to view the targets remaining hitpoints, distance as well as activating the targeting computer to compute where to aim to hit it with your weapons.
E Target Next Enemy
R Target Nearest Enemy
T Target Closest Unit to Crosshairs
F Target Next Friend
N Target Next Navigation Point

Wing Commands

Wing commands allow the player to control allied aircraft. Orders are placed on a stack: Giving a new order will over-ride any existing order, which will be reverted to once the new order has been completed.
F1 Follow Me
F2 Move to My Position
F3 Hold Current Position
F4 Attack Target
F5 Disregard Last Order
F6 Scatter (Disregard All Orders)
